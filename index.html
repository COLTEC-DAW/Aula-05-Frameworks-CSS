Idealizado e implementado por @mdo e @fat - dois desenvolvedores do Twitter - Bootstrap é um framework CSS com um propósito muito claro: ajudar desenvolvedores no processo de implementação de interfaces web.

Diferentemente de outros frameworks, tais como jQuery e Kendo UI, Bootstrap foi projetado para garantir um layout bonito, mas consistente de sua página web.

Dentre sua lista de recursos, podemos destacar:

  • Suporte ao layout responsivo
  • Grande leque de propriedades CSS
  • Suporte a ícones e componentes HTML customizados
  • Conjunto de bibliotecas JavaScript com as mais diversas funcionalidades (jQuery, Modais, Transições, etc.)

Atualmente, Bootstrap é o framework CSS mais utilizado entre os desenvolvedores web. Nesta apostila iremos demonstrar os principais recursos por trás deste grande framework.
